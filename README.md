Boundary Camel Example

# Configuration

## Boundary Configuration

The Boundary organization id and API keys is set by setting the following environment variables:

BOUNDARY_ORG_ID=
BOUNDARY_API_KEY=


## Running Camel

### Starting Camel


1. Set Boundary Organization and API key environmentvariables
2. Run the following:

```
$ ./camelrun.sh
```
